// Pipeline
01 ──
Image Input
ESP32-CAM stream, USB webcam, or local sample image.
02 ──
Classification
MobileNetV2 maps 1000 ImageNet labels to 3 waste categories.
03 ──
Sensor Fusion
Moisture, gas, weight, and fill-level data boost confidence.
04 ──
Servo Sort
Final category mapped to 0° / 90° / 180° servo angle.
05 ──
Dashboard
Flask + Socket.IO pushes live readings to the web UI.
// Features
🧠
Computer Vision
MobileNetV2 pretrained on ImageNet. No custom training required. TFLite variant for edge devices.
📡
Sensor Fusion
Bayesian update combining moisture, gas, load cell, and ultrasonic fill-level data.
⚙️
Servo Control
Simulated by default. Real hardware via Arduino serial or Raspberry Pi GPIO PWM.
📊
IoT Dashboard
Live Flask web dashboard with confidence gauges, sensor panels, and bin fill indicators.
🎥
Webcam Mode
Continuous live classification. Press C to classify, Q to quit.
🌍
Smart City Ready
Designed to extend to MQTT cloud platforms, mobile companion apps, and multi-bin arrays.
// Quick Start
# Clone and install
git clone https://github.com/krss-94/Smart-Waste-Segregation-And-Recycling-System.git
cd Smart-Waste-Segregation-And-Recycling-System
pip install -r requirements.txt

# Run demo cycle
python main.py --demo

# Launch IoT dashboard  →  http://localhost:5000
python main.py --dashboard

# Live webcam classification
python main.py --webcam
